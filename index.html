Beta Status

Stable for single-machine use. Expect breaking changes between beta versions.

  • Data Safety: Data lives in your Windows AppData folder.
  • Backups: You must manually export backups (Settings -> Export).
  • Sync: There is no cloud sync. One machine, one database.
