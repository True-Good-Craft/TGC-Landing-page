BUS Core · local-first workshop OS

Stop renting your own shop data.

BUS Core is a local-first “workshop OS” for makers and small shops. It tracks inventory, vendors, and runs on your own machine – no forced cloud, no monthly rent just to see your stock.

Built by a Canadian Forces materials tech with 12+ years in a real workshop, for people who actually touch materials.

How it’s being built

BUS Core is coded using an “AI as junior dev team” approach:

  • A strict Source-of-Truth text file defines how the system behaves.
  • Every change must pass a smoke test before it’s “real.”
  • The current build already runs a microkernel + SQLite core for items, vendors, and runs.
